< ul > Tag

Used in unordered list

Syntax
    < ul >
    ..list items..
    < /ul >

< ol > Tag

Used in ordered list

Syntax
    < ol >
    ..list items..
    < /ol >

< li > Tag

Used to enter list items

Syntax
    < li > ..list item.. < /li >

< dl > Tag

Used in detailed list

Syntax
    < dl >
    < dt >
    ..list items..
    < /dt >
    < dd >
    ..list items..
    < /dd >
    < /dl >

< img > Tag

Used to add the image:

    Syntax
    < img src="workplace.jpg" 
    alt="Workplace" usemap="#workmap" >

< map > Tag

Used to add the mapping:

    Syntax
    < map name="workmap" >

< area > Tag

Used to add the mapped area:

You must define the shape of the 
clickable area and you can 
choose one of these values:

rect - defines a rectangular region
circle - defines a circular region
poly - defines a polygonal region
default - defines the entire region
    Syntax
    < map name="workmap" >
    < area shape="rect" 
    coords="34, 44, 270, 350" 
    href="example.html" >

< input > Tag

Used to specify the type of input

< input type="text" > - 
    Displays a single-line text 
    input field


< input type="radio" > - 
    Displays a radio button 
    (for selecting one of many 
    choices) 


< input type="checkbox" > - 
    Displays a checkbox 
    (for selecting zero or more 
    of many choices)


< input type="submit" > - 
    Displays a submit button 
    (for submitting the form)


< input type="button" > - 
    Displays a clickable button

< label > Tag

Used to create a label for the input

The < label > tag defines a label for many form elements.

The 

Syntax

< form >

< label for="fname">First name:< /label>
< input type="text" id="fname" >
< label for="lname">Last name:< /label>
< input type="text" id="lname" >
< /form >
